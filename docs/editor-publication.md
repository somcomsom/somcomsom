# Publicació des de l’editor

L’editor publica `family.json`, `layout.json`, els dos fitxers d’overrides i el GEDCOM en un únic commit atòmic. Això evita desplegaments intermedis i garanteix que GitHub Pages validi sempre un estat complet.

La validació de layout comprova estructura, referències i coordenades numèriques, però no fixa posicions editables concretes. L’única relació visual preservada és l’alineació horitzontal entre `r53` i `r54`.
